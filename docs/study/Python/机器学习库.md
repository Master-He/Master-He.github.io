## numpy



numpy中矩阵乘法，星乘(*)和点乘(.dot)的区别  https://blog.csdn.net/wzyaiwl/article/details/106310705

```python
>>> import numpy as np
>>>
>>> a = np.array([[1,2],[3,4]])
>>> b = np.array([[5,6],[7,8]])
>>> a*b   # 对应元素相乘
array([[ 5, 12],
       [21, 32]])
>>> a.dot(b)  # 矩阵乘法
array([[19, 22],
       [43, 50]])
```
